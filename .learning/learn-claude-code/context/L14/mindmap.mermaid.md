# L14 Cron Scheduler Mindmap

```mermaid
mindmap
  root((Cron Scheduler))
    本质定位
      等开始 vs 等结果
        后台任务: 等结果
        定时调度: 等开始
      不是另一套 agent
        最终回到主循环
    三组件模型
      调度记录
        存储未来意图
        可持久化
      定时检查器
        后台独立循环
        分钟级检查
      通知队列
        统一注入机制
        主循环处理
    关键字段
      id
        唯一编号
      cron
        定时规则
        5字段: 分时日月周
      prompt
        到点执行的提示
      recurring
        是否反复触发
      durable
        是否落盘保存
        跨会话可用
      last_fired_at
        防重复触发
    实现要点
      检查周期
        分钟级而非秒级
        资源消耗考量
      通知注入
        时间到 → 放队列
        主循环统一处理
        不独立执行
    常见错误
      沉迷语法细节
      缺少 last_fired_at
      不支持落盘
      后台默默执行
      追求毫秒精度
    系统整合
      两条外部事件
        后台任务完成
        定时调度触发
      统一走通知队列
