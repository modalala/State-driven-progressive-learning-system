# Claude Harness 学习项目

本示例展示如何使用 progressive-learning-coach Skill 学习 Claude Code 的核心概念。

## 学习主题

**Claude 混合式 Harness 学习** - 结合 Claude 官方文档与 learn-claude-code 代码仓库，按先大纲后分节懒加载的方式学习 Claude API、tool use、skills、context 与 subagent 协作。

## 课程结构

| 课程 | 主题 | 预计时长 |
|------|------|----------|
| L0 | Claude 入门与 Agent 基本观 | 5 小时 |
| L1 | Tool Use 与 Harness 设计 | 7 小时 |
| L2 | Skills、上下文与多代理协作 | 6 小时 |

## 学习资源

`resources/` 目录包含：
- `code-snippets/` - 代码示例
- `documents/` - 相关文档
- `metadata.yaml` - 资源元数据

## 使用方法

```bash
cd examples/claude-harness-learning
# 对 AI 说："开始学习"
```

## 学习配置

本项目启用了所有三种学习方法：
- 方法1：心智模型建构 (20%)
- 方法2：结构化流程管理 (50%)
- 方法3：对抗性压力测试 (30%)
