## 对抗测试题库

### 题目 1: 如果把所有官方页面一次性加载进上下文，会更好吗？

**场景**: 你要设计一个学习助手，准备把所有 Claude Docs 页面正文一次性放进初始上下文。

**考点**: 上下文噪声、检索边界、按课懒加载的必要性

**脆弱点标记**: 资源组织误解

---

### 题目 2: 如果 agent 就是脚本，那 harness 还有什么意义？

**场景**: 有人认为一个写满流程控制的脚本已经等于 agent。

**考点**: 模型能力与工程环境的边界

**脆弱点标记**: 概念混淆
