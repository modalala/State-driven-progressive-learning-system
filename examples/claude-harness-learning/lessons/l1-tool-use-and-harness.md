## 对抗测试题库

### 题目 1: 既然有 web search，为什么还需要 web fetch？

**场景**: 有人认为只保留一个网络工具就够了。

**考点**: 工具职责边界与输入输出差异

**脆弱点标记**: 工具边界混淆

---

### 题目 2: 工具越多越好吗？

**场景**: 你在 harness 中打算暴露尽可能多的工具给模型。

**考点**: 复杂度、权限、安全边界、选择成本

**脆弱点标记**: 设计过度
